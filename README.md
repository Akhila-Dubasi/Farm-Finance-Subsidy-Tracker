## Farm Finance & Subsidy Tracker

A web-based application designed to help farmers efficiently manage crop-related financial data and track government subsidy information.

### Features
- Record and manage crop details and expenses
- Track government subsidies in a structured format
- Interactive dashboard for real-time insights
- Secure backend with real-time data updates

### Tech Stack
- Python
- Streamlit (Frontend & UI)
- Supabase (Backend & Database)

### Key Highlights
- Designed an intuitive user interface focused on simplicity and accessibility
- Implemented real-time data handling using Supabase
- Enables data-driven financial decision-making in the agriculture domain

### Use Case
Helps farmers and agricultural planners maintain transparent financial records and make informed decisions based on accurate data.
